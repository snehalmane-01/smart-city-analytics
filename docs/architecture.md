# Smart City Analytics Architecture

- Data Ingestion: adapters with validation and retries.
- Processing: stream window aggregations and anomaly detection.
- MLOps: model registry with versioning metadata.
- Serving: FastAPI prediction endpoints.
- Analytics: trend reports and comparative summaries.
- Infra: Docker, Kubernetes, Helm, Terraform.
- Observability: Prometheus, Grafana, alert rules, Jaeger config.
